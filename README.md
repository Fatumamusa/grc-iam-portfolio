## Why This Portfolio

Each project uses a single consistent fictional organization ("Meridian Health Partners," a regional healthcare system) so the work reads as one connected case study rather than disconnected exercises. Every artifact is built two ways where applicable: a manual/Excel version reflecting how most GRC and IAM teams operate day to day, and an automated version demonstrating lightweight scripting for governance tasks like access certification and risk scoring.
